### Hitboard Zero CAD 使用说明

![Hitboard成品图](./CAD.png)

#### 01 盖板

3-5mm 厚

放在顶层起到支撑手掌和展示壁纸的作用。一般采用透明亚克力或磨砂亚克力材质。

// 如果是亮色的壁纸，磨砂亚克力可以很好的柔光。如果是暗色的壁纸，透明亚克力可以让黑色更深沉。

#### 02 定位板

2mm 厚

14x14 的开孔，用于固定键盘轴体，并支撑敲击按键时的力。
// 中间有两组开孔，用户容纳螺丝的螺帽。

#### 03 轴座板

1.5mm 厚

用于支撑机械轴的轴座。将轴座插入每个按键的两小孔，并使用胶水固定。
// 中间两组螺丝孔，用于固定 pico。

#### 04 支撑板

2mm 厚

用于制造空腔，容纳轴座、pico、飞线。

// 顶部横梁默认是不断开的，提供更好的支撑性。如果是 typec 中央开孔，可以自行剪断。

#### 05 底部壁纸板

1mm 厚

用于支撑壁纸。//背面不放壁纸就不需要打印
