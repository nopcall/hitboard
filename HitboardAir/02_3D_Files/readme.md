#### 1. 文件说明

#### Hitboard Air 双版本 · 刚柔并济

**文件说明**

Hitboard Air 提供两种性格的版本——刚版与柔版。外壳、按键、Dock 三处的倒角设计，决定了设备整体的气质与手感。你可以根据自己的偏好，选择与你契合的那一款。

**刚 · 方正利落**
    
平整的极客体验。

    边缘干脆，线条直接。2.4mm 倒角仅做必要圆滑，保留直角带来的精准感。适合追求硬朗风格、喜欢清晰反馈的你。放在桌上，它是一块冷峻的金属感面板；握在手里，每一个棱角都在提醒你：这是一件趁手的工具。

**关键词：** 利落、精准、克制

**柔 · 圆润亲和**

温润的舒适体验。

    圆角过渡自然，握持更贴合手掌。每一个边缘都经过细致打磨，触感如鹅卵石般温润。适合长时间使用、偏爱柔和手感的你。它不会喧宾夺主，只是安静地承托你的每一次操作。

**关键词：** 圆润、舒适、包容


选择你的版本

刚与柔，没有好坏，只有不同。你的设备，该有你的性格。

**键帽触感**

按键根据键帽的朝向，有键帽朝上（正打）、键帽朝下（反打）两种方式。

正打能提供比较好的表面平整度，缺点是按键的确认感会不清晰。

反打按键的特点是，利用热床的颗粒，反打获得的表面随机颗粒效果，可以欺骗触觉（类似相机上的颗粒喷涂）。

// 注意：反打按键，需要在软件里把文件翻个朝向（也可以点击软件的「自动朝向」）。

##### ButtonsSystem

![HitboardAir 有刻键帽](./images/ButtonSystem2.0.JPG)

按键通过中框串联，不会掉的到处都是。能一定程度防止运输过程丢按键。每套按键都提供了一组独立的备用按键，包含 2 个功能键、1 个 23mm 按键，1 个 26mm 按键。

#### 2. 打印设置

** 支撑设置

01、02、04 板子，按默认面打印。00按键、03板子，需要在打印软件里翻个面。// 点击拓竹的自动翻面就可以

用不用支撑差别不大，可以开启树状支撑保险一点。

// 经常为了赶时间开狂暴模式打印，狂暴模式下出品质量也可以，因为基本都是垂直件没有悬浮。


** 朝向设置

04 外壳，倾斜 45 度打印，能让打印机的纹路横竖走，而不是 45度倾斜走。这样纹路会更方正、更有稳定感。。

00、01、02、03，用默认的朝向打印，45 度纹路，视觉上减少存在感。


#### 3. 打印材料

Hitboard Air 是基于 PETG 材料设计，有足够的强度和耐久度，计划长时间使用的话推荐使用 PETG 作为打印材料。

// 鉴于很多人的 3D 打印机只能打 PLA，对 PLA 也做了打印测试：精度、强度都还不错。长时间使用耐久度待观察。

// 注意！PLA 高温下容易形变，打印完成后，务必等板子冷却后，再取。否则可能会让板子弯曲。
