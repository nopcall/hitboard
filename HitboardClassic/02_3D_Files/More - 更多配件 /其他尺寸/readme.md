Hitboard Classic 尺寸选择

Classic 大杯
标准版（中杯）：256x174mm，摇杆边距74mm。
//特点是小巧，让人愿意拿起来玩。内部放钢珠可以配重到2.5KG。

加大版（大杯）：290x180mm，摇杆边距90mm。
//宽度更大，可配重到5KG。比较适合街机骨灰玩家。


作为摇杆，手掌不太胖的话，标准版宽度就够了。（图中手宽35mm，手长180mm）。

加大版整个手感会比较呆，缺乏操控感。没有特色，样子也不讨喜。如果不是自己有打印机可以低成本获取的话，不推荐。

两个型号，都支持A1mini打印机。

省流：标准版加凯华矮轴是本项目的完全体，建议以此配置入坑。如果不适合摇杆再转为Hitbox，除了30多块钱的摇杆其他零件都不会浪费。
